# Square-lattice self-avoiding walk connective constant $\mu_{\mathbb{Z}^2}$

## Description of constant

Let $\mathbb{Z}^2$ denote the **square lattice graph** with vertex set $\mathbb{Z}^2$ and edges between nearest neighbors (Euclidean distance $1$).

A **self-avoiding walk (SAW)** on a graph $G=(V,E)$ is a walk that visits no vertex more than once. In particular, for $\ell=1,2,\dots$ and $v\in V$, let $N(v,\ell)$ denote the number of SAWs in $G$ of length $\ell$ starting at $v$.
<a href="#SSSY2014-Nv-ell">[SSSY2014-Nv-ell]</a>

The **connective constant** (also called the SAW growth constant) of a graph $G$ is defined by
$$
\mu(G)\ :=\ \sup_{v\in V}\ \limsup_{\ell\to\infty} N(v,\ell)^{1/\ell}.
$$
<a href="#SSSY2014-def-cc">[SSSY2014-def-cc]</a>

For **vertex-transitive** graphs, the $\limsup$ in the definition above can be replaced by a true limit.
<a href="#SSSY2014-rem-vtx-limit">[SSSY2014-rem-vtx-limit]</a>

For the square lattice $G=\mathbb{Z}^2$, let $c_n$ be the number of $n$-step SAWs starting at the origin. Then the **square-lattice SAW connective constant** is
$$
C_{38} := \mu_{\mathbb{Z}^2}\ :=\ \lim_{n\to\infty} c_n^{1/n}.
$$

## Additional comments and links

- **Status (rigorous bounds vs. numerical estimate).** The exact value of $\mu_{\mathbb{Z}^2}$ is not known. <a href="#SlaBounds-unknown">[SlaBounds-unknown]</a>
  Moreover, <a href="#FV2017">[FV2017]</a> reports the rigorous interval
  $$
  2.625622\ <\ \mu_{\mathbb{Z}^2}\ <\ 2.679193,
  $$
  and also records a high-precision numerical estimate $\mu_{\mathbb{Z}^2}\approx 2.63815853032790(3)$.
  <a href="#FV2017-bounds-square">[FV2017-bounds-square]</a> <a href="#FV2017-estimate-square">[FV2017-estimate-square]</a> <a href="#FV2017-ref-180">[FV2017-ref-180]</a>

- **Historical origin of the notion.** A modern reference notes that “the first reference to the connective constant” appears in classical work of Hammersley and collaborators (1950s).
  <a href="#SSSY2014-history">[SSSY2014-history]</a>

- Surveys/background: <a href="#FV2017">[FV2017]</a>, <a href="#SSSY2014">[SSSY2014]</a>.
